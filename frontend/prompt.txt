## PROJECT OVERVIEW

This is an AI-powered learning assistant.

Users can:
- Register / Login
- Upload PDF documents
- PDFs are parsed, chunked, and processed asynchronously
- Generate Flashcards via Gemini AI
- Generate Quizzes via Gemini AI
- Study Flashcards
- Take Quizzes and view results
- Chat with the document (RAG)
- Ask AI to explain a specific concept from the document
- Generate document summaries
- Track learning progress via dashboard
- make starred cards count update in real-time

## AUTH API (USE AS-IS)

- POST /api/auth/register
- POST /api/auth/login
- GET  /api/auth/me

JWT is required for all protected routes.

---

## DOCUMENTS API (STRICT)

- POST   /api/documents/upload
  - multipart/form-data
  - fields: file (PDF), title
  - document status: processing | ready | failed

- GET    /api/documents
- GET    /api/documents/:id
- PUT    /api/documents/:id
- DELETE /api/documents/:id

Each document includes:
- flashcardCount
- quizCount

---

## FLASHCARDS API

- GET    /api/flashcards
- GET    /api/flashcards/:documentId
- POST   /api/flashcards/:cardId/review
- PUT    /api/flashcards/:cardId/star
- DELETE /api/flashcards/:id

Flashcards are GENERATED via Gemini (not static).

## QUIZZES API (IMPORTANT – USE EXACT ROUTES)

- GET    /api/quizzes/:documentId
- GET    /api/quizzes/quiz/:id
- POST   /api/quizzes/:id/submit
- GET    /api/quizzes/:id/results
- DELETE /api/quizzes/:id

Submit quiz payload:

## AI / GEMINI API (VERY IMPORTANT)

These endpoints trigger Gemini AI on the backend.
Frontend MUST provide correct payloads and UI flows.

### Generate Flashcards
- POST /api/ai/generate-flashcards
Body:
{
  documentId,
  count
}

### Generate Quiz
- POST /api/ai/generate-quiz
Body:
{
  documentId,
  numQuestions,
  title
}

### Generate Summary
- POST /api/ai/generate-summary
Body:
{
  documentId
}

### Chat with Document (RAG)
- POST /api/ai/chat
Body:
{
  documentId,
  question
}

### Explain Concept
- POST /api/ai/explain-concept
Body:
{
  documentId,
  concept
}

### Get Chat History
- GET /api/ai/chat/:documentId
